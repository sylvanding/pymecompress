## 核心概念速查

### 什么是PYMECompress？

PYMECompress是一个专为**光子噪声限制图像**设计的压缩库，主要用于荧光显微镜等科学成像领域。

### 核心特点

- ✅ **科学保真**: 压缩误差控制在泊松噪声范围内
- ⚡ **高性能**: 单核吞吐量500-600 MB/s
- 🔧 **易用**: 提供多层次API，从简单到精细控制
- 📦 **兼容**: 支持numcodecs和Zarr生态系统

### 主要算法

1. **量化**: 平方根变换，保持误差在噪声内
2. **霍夫曼编码**: 无损压缩，接近熵极限
3. **AVX优化**: SIMD并行加速

### 典型用例

```python
# 最简单的使用方式
from pymecompress import codecs
import numpy as np

# 准备数据（光子计数图像）
data = np.random.poisson(100, (512, 512)).astype('uint16')

# 压缩
codec = codecs.HuffmanQuant16(offset=0, scale=1.0)
compressed = codec.encode(data.ravel())

# 解压
decompressed = codec.decode(compressed).reshape(data.shape)

# 压缩比
print(f"压缩比: {data.nbytes / len(compressed):.2f}x")
```
