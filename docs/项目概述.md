# PYMECompress 项目概述

## 项目简介

PYMECompress 是一个专为光子噪声限制图像设计的压缩库，它能够在泊松噪声范围内保持损失可控。该项目主要用于显微镜成像等科学计算领域，可以实现约500-600MB/s的单核吞吐量。

## 核心功能

1. **霍夫曼编码（Huffman Coding）**: 基于BCL (Basic Compression Library)的高度优化版本
2. **AVX优化量化器**: 针对光子限制图像进行"噪声范围内"的量化
3. **Python接口**: 提供易用的Python API
4. **numcodecs编解码器**: 支持与Zarr等IO包的无缝集成

## 技术栈

- **语言**: C、Cython、Python
- **主要依赖**: NumPy、Cython
- **编译器**: GCC/Clang（需要AVX扩展支持）
- **构建系统**: setuptools、meson

## 安装方式

1. **conda安装** (推荐):
   ```bash
   conda install -c david_baddeley pymecompress
   ```

2. **pip安装**:
   ```bash
   pip install pymecompress
   ```

3. **源码安装**:
   ```bash
   python setup.py install
   ```
   注意：Windows平台需要使用mingw32编译器

## 性能特点

- 单核吞吐量: 500-600 MB/s
- 损失控制在泊松噪声范围内
- AVX指令集优化
- 适用于科学成像数据压缩
